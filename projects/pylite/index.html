Pré requis

Python 3

Utilisez la commande ci-dessous pour installer Python 3.
$ sudo apt-get install python3

disco-py

Utilisez la commande ci-dessous pour installer disco-py
$ git clone https://github.com/b1naryth1ef/disco

Python GTK+ 3

Utilisez la commande ci-dessous pour installer Python GTK+ 3
$ sudo apt-get install python3-gi
