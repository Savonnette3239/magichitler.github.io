Application Magic Hitler
Application graphique Python destinée à configurer et utiliser un bot de raid Discord avec une facilité déconcertante.
Le script utilisé par l'interface et le bot sont en JavaScript, et la librairie utilisée est Discord.js.
Son avancement est pour l'instant arrêté, faute de motivation, mais vous pouvez postuler à son amélioration dans l'onglet "Nous contacter".

Langages : JavaScript, Python
Date de sortie : 9 mai 2018
Version actuelle : 1.4.5
Développement : Suspendu depuis le 20 septembre 2018
Requis : Python 3, Node.js
